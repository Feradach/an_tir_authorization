Welcome to the test version of the new Authorization Portal for An Tir. During the development process, you can send feedback to Feradach at viperzka@gmail.com. Please use "Authorization Portal" in the subject line.

If you have encountered a bug, you can use the form below to report it. If you have a screenshot of the bug, please include it in the form.

Report a Bug

The database for this test is filled with random users and authorizations so that you can test the portal. During the test, you will be able to create your own account using the button below. After creating an account you will be able to add authorizations and set yourself as a marshal officer on your account page. These features will not exist in the final version of the portal.
